# ChessMark Icons

Source and generated icon files for the Chrome extension.

- `icon_source.svg`: editable source icon
- `icon_16.png`: toolbar and small browser UI contexts
- `icon_32.png`: high-density toolbar contexts
- `icon_48.png`: extension management page
- `icon_128.png`: install surfaces
- `store_icon_128.png`: Chrome Web Store listing icon with transparent padding

The icon combines a simplified chessboard and yellow bookmark ribbon to communicate saving chess positions for later study.
